##  Sobre o projeto

O **Comuniq** é uma plataforma criada para ajudar trabalhadores autônomos da comunidade a conseguirem mais clientes de forma simples e acessível.

A ideia é conectar:

*  Prestadores de serviço (pedreiros, manicures, eletricistas, etc.)
*  Pessoas que precisam desses serviços

Tudo isso sem complicação, sem login e com contato direto via WhatsApp.

---

##  Objetivo

Facilitar a divulgação de serviços locais e reduzir a dependência do “boca a boca”, oferecendo um catálogo digital simples e eficiente.

---

##  Funcionalidades

✔ Cadastro de prestadores sem necessidade de senha
✔ Busca por:

* Nome
* Categoria
* Bairro
* ID

✔ Sistema de avaliação  (1 a 5 estrelas)
✔ Contato direto via WhatsApp
✔ Identificação de prestadores verificados (com CNPJ)
✔ Interface simples e responsiva (funciona no celular)

---

##  Como funciona

O site funciona como uma **lista de contatos inteligente**, onde:

1. O prestador se cadastra
2. O sistema salva os dados
3. O cliente encontra o profissional
4. O contato acontece via WhatsApp

---

##  Tecnologias utilizadas

* HTML → estrutura da página
* CSS → design e responsividade
* JavaScript → lógica e funcionalidades

---

## Armazenamento de dados

Os dados são salvos no **localStorage do navegador**, ou seja:

*  Não usa banco de dados
*  Não precisa de servidor
*  Funciona diretamente no navegador

---

## Entrada de dados (Input)

O usuário preenche informações como:

* Nome
* Tipo de serviço
* Descrição
* Bairro
* WhatsApp
* Telefone
* CNPJ (opcional)

---

## Saída de dados (Output)

O sistema exibe:

* Lista de profissionais (cards)
* Nome e serviço
* Bairro
* Avaliação 
* Botão de contato via WhatsApp
* ID automático (ex: CQ-00001)

---

## Diferenciais

* Simples e fácil de usar
* Não exige login
* Focado em acessibilidade para a comunidade
* Rápido e leve

---

## Resumo

O **Comuniq** é uma solução digital simples que ajuda trabalhadores a divulgarem seus serviços e clientes a encontrarem profissionais de forma rápida e prática.

---

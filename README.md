
# Comuniq 🚀 | Conectando a Comunidade

![Status](https://img.shields.io/badge/status-Beta-yellow.svg)
![Tecnologia](https://img.shields.io/badge/stack-Vanilla%20JS-orange.svg)
![IA](https://img.shields.io/badge/IA-Gemini%20%26%20Claude-blue.svg)

O **Comuniq** é uma plataforma comunitária projetada para conectar moradores de periferias de São Paulo a prestadores de serviços locais, reduzindo a dependência do "boca a boca" através de um catálogo digital acessível.

## 📋 Funcionalidades Principais

* **Busca Inteligente (Tobias IA):** Sistema que interpreta linguagem natural para encontrar profissionais.
* **Contato Direto:** Integração com WhatsApp para negociações diretas.
* **Selo Verificado:** Identificação para prestadores que fornecem CNPJ.
* **Moderação por IA:** Filtro automático de ofensas em avaliações para manter o respeito na comunidade].

| **Chat Tobias** | Gemini Flash | Atendimento geral e acolhedor. |

| **Moderação** | Gemini Flash | Bloqueio de conteúdo ofensivo. |
| **Gerador de Perfil** | Gemini Flash | Escrita de descrições profissionais. |

---

## 🛠️ Detalhes Técnicos

* **Stack:** HTML5, CSS3 (Mobile-first) e JavaScript Vanilla.
* **Persistência:** Uso de `localStorage` para salvar dados no navegador.
* **Modelos:** Integração via API com Google Gemini e Anthropic Claude.
